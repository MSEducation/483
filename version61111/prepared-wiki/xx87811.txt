The following is a list of games for the Game Boy Color.  All games are released in Europe and the USA. The number of games in this list is 469, organized alphabetically by name. See Lists of video games for related lists.

==Unlicensed games==

Several unlicensed games were released for the Game Boy Color.  Almost all are backward compatible with older Game Boy systems.  Titles consist of Christian video games as well as counterfeit games based on popular franchises.
