[[WuGeo]]

CATEGORIES: Python libraries, Earth sciences graphics software

WuGeo is a Python library
used to draw points on geographical maps, starting from latitude and longitude.
It borns from the desire to track site visitors, then becomes an abstract library, being
a scientific generic tool.
In particular, WuGeo has to be used with world maps created with the Miller cylindrical projection,
that is a modified Mercator projection.
Library Audience: developers
