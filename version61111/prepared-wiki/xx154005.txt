[[Inmate Code]]

CATEGORIES: Imprisonment and detention

Inmate Code (sometimes called "Convict Code") refers to the rules and values that have developed among prisoners inside prisons social systems.[tpl]cite book |author=Clear, Todd R., Cole, George F., Resig, Michael D.|year=2006 |title=American Corrections Seventh Edition |publisher=Thompson[/tpl] The inmate code helps define an inmate's image as a model prisoner. The code helps to emphasize unity of prisoners against correctional workers. 
This code highlights the reasons why "rehabilitation" is often so difficult to achieve.
