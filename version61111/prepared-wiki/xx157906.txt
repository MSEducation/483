[[ProjectWise]]

CATEGORIES: Computer-aided design software, Computer-aided design software for Windows, Content management systems, Project management software

ProjectWise is a suite of engineering project collaboration software from Bentley Systems  designed for the architecture, engineering, construction (AEC) industries.[tpl]cite web|title=ProjectWise Project Team Collaboration Software Products|url=http://www.bentley.com/en-US/Products/projectwise+project+team+collaboration/| accessdate=2011-11-30 [/tpl] It helps project teams manage, share and distribute engineering project content and review in a single platform. While ProjectWise can manage any type of CAD, BIM, geospatial and project data, it integrates with Bentley applications, and other products including Autodesk software and Microsoft Office.[tpl]cite web | url=http://www.bentley.com/en-US/Products/ProjectWise+Integration+Server/Product-Overview.htm | title=ProjectWise Integration Server Overview| accessdate=2011-11-30 [/tpl]
[tpl]cite web | url= http://www.canadianconsultingengineer.com/news/software-toronto-seminar-reviews-bentley-systems-projectwise-tool/1000653249/| title=Software: Toronto seminar reviews Bentley Systems' ProjectWise tool [/tpl]
Content publishing tools
Design review tools
Mobile applications
Lifecycle management tools
Noted features include real-time collaboration as well as desktop or online deployment.
