[[HotRuby]]

CATEGORIES: Ruby programming language

HotRuby is a JavaScript and ActionScript implementation of the Ruby programming language. HotRuby runs Ruby source code on a web browser and Flash. A similar implementation is RubyJS, which performs better than HotRuby.http://www.ntecs.de/blog/articles/2008/03/26/rubyjs-vs-hotruby/
