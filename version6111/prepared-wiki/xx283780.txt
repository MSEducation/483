[[Web Start Today]]

CATEGORIES: Web design, Web hosting, Free web hosting services, Web development software, Internet companies of the United States, Companies based in Plainsboro, New Jersey, Privately held companies based in New Jersey

Web Start Today is a free website builder tool with several additional features, including SEO tools, web hosting and live tech support. It employs an intuitive interface, allowing the user to incorporate widgets through a drag and drop system. Web Start Today also integrates responsive templates, Web Start Today Web Shop (an online shop option to website builders) and acts as a domain registrar.

==Overview==

Web Start Today is a free online website builder, requiring an Internet connection from its user and storing all website projects on the site's server (it provides unlimited storage and bandwidth).[tpl]cite web | title = Web Start Today: How It Can Help Your Business Grow Online | publisher = Explore B2B| date = 18 June 2013 | url = https://exploreb2b.com/articles/web-start-today-how-it-can-help-your-business-grow-online | accessdate = 24 August 2013[/tpl] It is built under a Content Management System (CMS) framework, working on the intuitive What You See Is What You Get (WYSIWYG) principle (the user can see how the finished work will look like before publishing it). There are two admin panels within the CMS: the WYSIWYG editor and the Manage Pages panel. In the former the user can make changes directly to the content of the pages, while in the latter they can edit the settings of the website pages.[tpl]cite web | title = WebStartToday Review | publisher = Superb Website Builders| url = http://superbwebsitebuilders.com/webstarttoday-review/ | accessdate = 24 August 2013[/tpl]
Web Start Today has a range of professionally designed templates, categorized per industry (72 business specific verticals),[tpl]cite web | title = Web Start Today  | publisher = Download.com| url = http://download.cnet.com/Web-Start-Today/3000-10248_4-75940353.html | accessdate = 24 August 2013[/tpl] with SEO, e-commerce and security features. It also offers HTML5 and CSS3 responsive templates, i.e. the website design and layout will adjust to the viewer’s device screen resolution.[tpl]cite web | title = Web Start Today: The Simplest Ever Website Builder  | date =  22 May 2013| url = http://marketingchris.com/web-start-today-the-simplest-ever-website-builder/ | accessdate = 24 August 2013[/tpl]
After registration, the template chosen by the user can be customized by choosing color themes, adding text, images, logo, multimedia and by managing the settings of each subpage.[tpl]cite web | title = Creating A Free Bookie Site With WebStartToday in 5 Steps | date = 18 June 2013 | url = http://www.priceperhead.com/tips/creating-free-bookie-site-webstarttoday/ | accessdate = 24 August 2013[/tpl][tpl]cite web | title = Launch Your Freelance Website in a Day – Using WebStartToday| date = 8 May 2013 | url = http://freelanceswitch.com/start/site-launch-with-webstarttoday/ | accessdate = 24 August 2013[/tpl] The templates include pre-populated industry-specific SEO content that can be edited and personalized, if required.
The user can publish the website for free as a sub-domain of WebStartToday.com or use the paid option, which provides a personal web domain and domain based email accounts. The paid option is monthly or yearly and there are no different packages in terms of feature-richness.  Web Start Today also offers some paid add-ons, like Hire a Pro (one-time fee for a web master to help build the website), SEO Package (monthly fee for search engine optimisation), Custom Content Creation (research for improving the page content, fee based on the number of pages).[tpl]cite web | title = Web Start Today: Paid Website Builders Review | publisher = IRDB.com| url = http://www.irdb.com/website-builders/webstarttoday/ | accessdate = 24 August 2013[/tpl]
