[[Network intrusion detection system]]

CATEGORIES: Internet Protocol based network software, Computer network security, Intrusion detection systems

In computer security, a Network Intrusion Detection System (NIDS) is an intrusion detection system that attempts to discover unauthorized access to a computer network by analyzing traffic on the network for signs of malicious activity.Ptacek, Thomas H. & Newsham, Timothy N. (January 1998); "Insertion, Evasion, and Denial of Service: Eluding Network Instrusion Detection"
