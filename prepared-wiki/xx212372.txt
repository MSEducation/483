[[Altibase]]

CATEGORIES: Proprietary database management systems, Software companies of South Korea, Database companies, Companies established in 1999, 1999 establishments in South Korea

Altibase founded in 1999, is a relational database management system (RDBMS) vendor that specializes in In-Memory computing technology.  Altibase focuses on high-intensity transactional processinghttp://www.gartner.com/document/2570415 through RAM memory, complementing its disk-resident database in a single database solution.http://www.storagenewsletter.com/rubriques/software/altibase-hdb-hybrid-database-management/

==Products and Services==

===Altibase XDB™===

ALTIBASE XDB™ is a traditional relational database that has fast data processing speeds in main-memory. It comes with High-Availability, replication and scalability features. http://support.altibase.com/en/product ALTIBASE XDB™ comes with 3 interfaces that include Direct Access Mode, Direct Access API Mode as well as conventional client/server protocols such as TCP/IP and IPC for more complex database operations.http://www.ciokorea.com/news/17349  Gartner's Who's Who for In-Memory DBMSs 2014 indicates: ALTIBASE XDB™ supports full ACID properties, standard connectivity interfaces such as JDBC and ODBC supported as well as interoperability. https://www.gartner.com/doc/2650915/whos-inmemory-dbmss

===Altibase HDB™===

ALTIBASE HDB is a "hybrid DBMS with Hybrid Architecture" that combines an in-memory database with a conventional disk-resident database.http://atc.altibase.com/sub09/611b/html/Admin/ ALTIBASE HDB™ features include SQL 99 standards compliance, High-Availability, replication, real-time ACID compliance, spatial, vertical and horizontal scalability,  and compatibility with standard interfaces and programming languages such as ODBC, JDBC, OLE DB, .Net, Java, C/C++, Precompiler.
Gartner notes, in addition to applying its technology to unique billing scenarios in telecom, as well as real-time flaw detection in manufacturing scenarios, customers cite their use of HDB for analytics and storage of text and rich media content.http://oracledbacr.blogspot.com/2014/01/magic-quadrant-for-operational-database.htmlhttps://www.gartner.com/doc/2650915/whos-inmemory-dbmss
According to Gartner, Altibase's hybrid database (HDB) is based on SQL and is fully ACID compliant. HDB can run both in-memory and on physical disk which "...allows users to manage data with different SLA's..." https://www.gartner.com/doc/2650915/whos-inmemory-dbmss

===Altibase CEP™===

ALTIBASE CEP™ is a Windows-based data caching software for use in real-time Complex event processing applications.http://www.itdaily.kr/atl/view.asp?a_id=35647 This product was formerly known as DSM ("Altibase Data Stream").
