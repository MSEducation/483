==Decoding Software==

It is possible to decode morse code using software. The variety ranges from wide-band software defined radio receivers coupled to the Reverse Beacon NetworkReverse Beacon Network http://www.reversebeacon.net/ where a software for the Windows operating system decodes a bunch of signals at onces and detects CQ messages on ham bands, to mobile apps (i.e. for the iPad Comparison of morse decoder apps for the iPad http://gerolfziegenhain.wordpress.com/2013/05/20/morse-decoder-test-iphone-ipad/).
