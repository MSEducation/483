[[XNA Racer]]

CATEGORIES: Microsoft games, Racing video games, Windows games, Xbox 360 Live Arcade games

XNA Racer, also known as Racing Game, is a game distributed by Microsoft as a starter kit for XNA Game Studio developers.  Developers can download the entire set of source code and assets needed to build and modify the game.
XNA Racer will run on PC computers running Windows XP or Windows Vista; and XNA Creators Club members can also build and run the game on Xbox 360 consoles.
