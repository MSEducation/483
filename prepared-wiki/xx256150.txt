[[C12G]]

CATEGORIES: Software companies of Spain, Cloud infrastructure

C12G Labs is an enterprise cloud computing software company which provides OpenNebula-based software and services.R. S. Montero, I. M. Llorente. "OpenNebula: Leading Innovation in Cloud Computing Management",  ERCIM News, October 2010) It manages and contributes to the OpenNebula open-source project, and provides an OpenNebula toolkit distribution for the enterprise called OpenNebulaPro.[tpl]cite web|url=http://www.c12g.com/products|title=What is OpenNebulaPro?|accessdate=30 December 2010[/tpl] C12G (numeronym for "cloud computing") was founded in April 2010[tpl]cite web |url=http://blog.opennebula.org/?p=509 |title=OpenNebula Cloud Toolkit Goes Commercial |accessdate=27 December 2010[/tpl][tpl]cite web |url=http://www.the451group.com/report_view/report_view.php?entity_id=62558 |title=451 Group report's "C12G commercializes OpenNebula for cloud creation in the enterprise" |accessdate=27 December 2010[/tpl] and is a company associated with the Scientific Park of Madrid.[tpl]cite web |url=http://www.fpcm.es/empresasincubadas08octeng.htm |title=Companies in the Scientific Park of Madrid |accessdate=27 December 2010[/tpl]
